# 例题 1.3 模型

## 题意重述
求解如下线性规划问题：

最大化：

- z = 2x1 + 3x2 - 5x3

约束条件：

- x1 + x2 + x3 = 7
- 2x1 - 5x2 + x3 >= 10
- x1 + 3x2 + x3 <= 12
- x1, x2, x3 >= 0

## 决策变量
- x1, x2, x3：决策变量（非负）

## 目标函数
最大化：

- 2x1 + 3x2 - 5x3

- 等式约束：x1 + x2 + x3 = 7
- 不等式约束：2x1 - 5x2 + x3 >= 10
- 不等式约束：x1 + 3x2 + x3 <= 12
- 非负性：x1, x2, x3 >= 0

# 例题 1.8 模型

## 题意重述
目标是最小化最大误差：

min max |epsilon_i|，其中 epsilon_i = x_i - y_i。

为便于求解，取一组具体数据：
- t = [1, 2, 3, 4, 5]
- y = [1.2, 2.0, 2.7, 3.9, 5.1]

并约束拟合值满足直线关系：x_i = a + b * t_i。

## 决策变量
- a, b：直线参数
- x_i：在 t_i 处的拟合值
- z：最大绝对误差（z >= 0）

## 目标函数
最小化 z。

## 约束条件
- x_i = a + b * t_i，forall i
- x_i - y_i <= z，forall i
- y_i - x_i <= z，forall i

以上两条不等式对 |x_i - y_i| 进行线性化，使得 z = max_i |x_i - y_i|。

## 说明
Gurobi 支持 abs/max 的一般约束，但参考解采用线性化形式，便于移植到其他求解器。

## 理解与解释
“最优直线问题”可以理解为：给定一组散点 (t_i, y_i)，用一条直线 x = a + b * t 去代表它们，选择 a、b 使“误差最小”。“误差最小”的定义不同，对应不同模型，例如最小二乘、最小绝对偏差以及切比雪夫逼近（最小化最大偏差）。

本题的形式是最小化 max_i |x_i - y_i|，并加入约束 x_i = a + b * t_i，使所有拟合值共线。于是问题转化为：在所有可能的直线中，找到一条使“最远点到直线的垂直距离”最小的直线，这正是切比雪夫逼近的最优直线问题。

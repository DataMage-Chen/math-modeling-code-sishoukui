# 数学建模算法与应用（第3版）复现项目

本项目使用 Python + Gurobi 复现《数学建模算法与应用（第3版）》中的例题与习题。

参考书籍：
- 司守奎、孙玺菁 编著：《数学建模算法与应用（第3版）》
- 司守奎、孙玺菁 编著：《数学建模算法与应用（第3版）习题解答》

## 项目目录结构

```
.
├─ ch01/                # 第 1 章（后续章节按需创建 ch02 ~ ch16）
│  ├─ ex01_01/           # 例题 1.1
│  └─ hw01_01/           # 习题 1.1
└─ README.md
```

## 题目目录命名规则

- 例题：`ex{章号}_{题号}`，例如 `ex01_01` 表示 例题 1.1
- 习题：`hw{章号}_{题号}`，例如 `hw01_01` 表示 习题 1.1

## 每道题的最小目录结构

```
ex01_01/
├─ model.md      # 题意重述、变量定义、约束条件、目标函数
├─ solution.py   # Gurobi 主程序
├─ data/         # 原始数据（若有）
└─ results/      # 输出结果、日志、图表
```

说明：后续章节与题目均遵循同样的结构与命名规则。
